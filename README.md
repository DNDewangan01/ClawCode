# ClawCode Advanced

A modern code editor with advanced GUI features built with PyQt5.

## Features

- 📁 **File Explorer** - Browse and open files directly from the sidebar
- 📑 **Multi-tab Editing** - Work with multiple files simultaneously
- 🎨 **Dark Theme** - Easy on the eyes with VS Code-inspired dark theme
- 🔍 **Zoom Controls** - Zoom in/out for better readability
- 💾 **Save/Open Files** - Full file management support
- ✏️ **Edit Operations** - Undo, Redo, Cut, Copy, Paste, Select All
- 🚀 **Run Button** - Placeholder for code execution (coming soon)

## Installation

```bash
# Install Python dependencies
pip install -r requirements.txt
```

## Usage

```bash
# Run the GUI application
python3 clawcode_gui.py
```

## Keyboard Shortcuts

- `Ctrl+N` - New file
- `Ctrl+O` - Open file
- `Ctrl+S` - Save file
- `Ctrl+Shift+S` - Save As
- `Ctrl+Q` - Exit
- `Ctrl+Z` - Undo
- `Ctrl+Y` - Redo
- `Ctrl+X` - Cut
- `Ctrl+C` - Copy
- `Ctrl+V` - Paste
- `Ctrl+A` - Select All
- `Ctrl++` - Zoom In
- `Ctrl+-` - Zoom Out
- `Ctrl+0` - Reset Zoom

## Requirements

- Python 3.8+
- PyQt5
